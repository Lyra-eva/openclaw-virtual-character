## 🚀 使用方法

### 方式 1: UI 界面

1. **打开模型选择器**
   - 点击底部 🎨 按钮

2. **选择模型**
   - 浏览预设列表
   - 点击模型卡片

3. **自定义颜色**
   - 选择分类标签
   - 调整颜色滑块
   - 点击"应用自定义颜色"

4. **保存配置**
   - 系统自动保存
   - 下次启动自动加载

---

### 方式 2: 代码调用

```typescript
import { avatarModelService } from './services/AvatarModelService';

// 1. 获取所有预设
const presets = avatarModelService.getPresets();

// 2. 切换模型
await avatarModelService.switchToPreset('anime-female', scene);

// 3. 自定义颜色
avatarModelService.customizeCurrentModel({
  hair: { color: '#ff69b4' },
  eyes: { color: '#00bfff' },
  skin: { color: '#ffe4e1' },
});

// 4. 添加自定义预设
avatarModelService.addCustomPreset({
  id: 'my-custom-model',
  name: '我的自定义模型',
  modelUrl: '/models/my-avatar.glb',
  category: 'custom',
  customizations: {
    hairColor: '#ff0000',
    eyeColor: '#00ff00',
  },
});
```

## 📦 模型导入流程

### 从 Ready Player Me 导入

1. **访问网站**
   ```
   https://readyplayer.me/
   ```

2. **创建角色**
   - 上传照片或手动设计
   - 调整发型/眼睛/皮肤等
   - 完成设计

3. **下载模型**
   - 点击"Download"
   - 选择 GLB 格式
   - 保存到 `public/models/`

4. **添加到预设**
   ```typescript
   avatarModelService.addCustomPreset({
     id: 'my-rpm-avatar',
     name: '我的 RPM 角色',
     modelUrl: '/models/my-avatar.glb',
     category: 'realistic',
   });
   ```

---

### 从 VRoid Studio 导入

1. **设计角色**
   - 打开 VRoid Studio
   - 创建 3D 动漫角色
   - 调整各项参数

2. **导出模型**
   - File → Export → Export as GLB
   - 或 Export as VRM

3. **转换为 GLB** (如果是 VRM)
   ```bash
   # 使用 three-vrm 转换
   npm install @pixiv/three-vrm
   ```

4. **导入应用**
   ```typescript
   await avatarModelService.loadModel({
     url: '/models/vroid-avatar.glb',
     scale: 1,
   });
   ```

## 🛠️ 高级自定义

### 材质修改

```typescript
import * as THREE from 'three';

// 修改材质属性
gltf.scene.traverse((child) => {
  if ((child as THREE.Mesh).isMesh) {
    const mesh = child as THREE.Mesh;
    const material = mesh.material as THREE.MeshStandardMaterial;
    
    // 修改颜色
    material.color.set('#ff0000');
    
    // 修改粗糙度
    material.roughness = 0.5;
    
    // 修改金属度
    material.metalness = 0.8;
    
    // 修改透明度
    material.transparent = true;
    material.opacity = 0.8;
    
    material.needsUpdate = true;
  }
});
```

## 🐛 常见问题

### Q: 模型加载失败？

**A:** 
- 检查模型路径是否正确
- 确保是 GLB/GLTF 格式
- 检查文件大小 (<50MB)
- 查看控制台错误信息

### Q: 颜色不生效？

**A:**
- 检查材质名称是否匹配
- 确保材质支持颜色修改
- 调用 `material.needsUpdate = true`

### Q: 性能卡顿？

**A:**
- 降低模型面数
- 减小贴图大小
- 减少材质数量
- 使用 Draco 压缩
