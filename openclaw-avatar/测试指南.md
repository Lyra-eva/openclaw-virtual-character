## 📋 测试清单

### 1. 初始化测试

```typescript
import { ultimateVoiceService } from './services/UltimateVoiceService';

await ultimateVoiceService.initialize({
  enableMicroExpressions: true,
  enableEmotionalTTS: true,
  enablePredictiveLipSync: true,
  enableBodyLanguage: true,
  enableContextAware: true,
});

const status = ultimateVoiceService.getStatus();
console.log(status);
// 应显示：isInitialized: true
```

**预期结果：**
- ✅ 控制台显示 "✅ All services initialized"
- ✅ `status.isInitialized === true`

---

### 2. 微表情测试

```typescript
import { microExpressionService } from './services/MicroExpressionService';

// 测试开心表情
const units = microExpressionService.generateFromEmotion('happy', 0.8);
microExpressionService.applyActionUnits(units);

const state = microExpressionService.update(0.016);
console.log('瞳孔:', state.pupilSize);
console.log('眨眼:', state.blinkAmount);
```

**预期结果：**
- ✅ AU12 (嘴角拉起) > 0.6
- ✅ AU06 (脸颊抬起) > 0.4
- ✅ 瞳孔大小 ≈ 3.6mm (开心时放大)

### 3. 情感语音测试

### 4. 手势测试

```typescript
import { bodyLanguageService } from './services/BodyLanguageService';

bodyLanguageService.initialize();

// 测试挥手
const gesture = bodyLanguageService.selectGesture('你好呀');
console.log('选择的手势:', gesture?.name);

await bodyLanguageService.playGesture(gesture);

const state = bodyLanguageService.update(0.016);
console.log('左手位置:', state.leftHand);
```

**预期结果：**
- ✅ 选择到 "挥手" 手势
- ✅ 手部位置随时间变化
- ✅ 姿态自动切换到 "speaking"

---

### 5. 唇形同步测试

```typescript
import { predictiveLipSyncService } from './services/PredictiveLipSyncService';

const sequence = await predictiveLipSyncService.precomputeVisemes(
  '你好，这是测试',
  1.0
);

console.log('嘴型数量:', sequence.visemes.length);
console.log('总时长:', sequence.totalTime);

const viseme = predictiveLipSyncService.getPredictedViseme(
  sequence,
  0.5
);
console.log('0.5s 时的嘴型:', viseme.mouthOpen);
```

**预期结果：**
- ✅ 生成 10-20 个嘴型
- ✅ 总时长 ≈ 2-3 秒
- ✅ 嘴型开合度 0-1 之间

### 7. 完整集成测试

```typescript
import { ultimateVoiceService } from './services/UltimateVoiceService';

// 初始化
await ultimateVoiceService.initialize();

// 完整情感表达
await ultimateVoiceService.speakWithFullExpression(
  '哇！今天真是太开心了！',
  'excited'
);

// 每帧更新
function animate(deltaTime) {
  const state = ultimateVoiceService.update(deltaTime);
  
  // 验证状态
  console.log('嘴型:', state.mouthOpen);
  console.log('瞳孔:', state.pupilSize);
  console.log('手势:', state.leftHand, state.rightHand);
  
  requestAnimationFrame(animate);
}
animate(0.016);
```

**预期结果：**
- ✅ 微表情自动应用
- ✅ 嘴型精确同步
- ✅ 手势自然播放
- ✅ 呼吸模拟工作
- ✅ 所有数据实时更新

## 🐛 常见问题排查

### 问题 1: 初始化失败

**错误信息：**
```
Error: 初始化失败：模型文件不存在
```

**解决方法：**
```bash
# 下载模型
npm run download-models

# 检查模型文件
ls -lh public/models/
```

---

### 问题 2: 微表情无效果

**检查代码：**
```typescript
const state = microExpressionService.getStatus();
console.log(state);

// 如果 actionUnits 全为 0
// 检查是否调用了 applyActionUnits
```

---

### 问题 3: 手势不播放

**检查：**
```typescript
const gesture = bodyLanguageService.selectGesture(text);
console.log('选择的手势:', gesture);

// 如果为 null
// 检查文本是否包含关键词
```

---

### 问题 4: 唇形不同步

**调整：**
```typescript
// 调整预测时间
predictiveLipSyncService.setLookaheadTime(100); // 100ms

// 检查预计算
const sequence = await predictiveLipSyncService.precomputeVisemes(text);
console.log('嘴型数量:', sequence.visemes.length);
```
